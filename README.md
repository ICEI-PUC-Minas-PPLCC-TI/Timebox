# Administração do Tempo 3 

Objetivo: Plataforma web onde as pessoas conseguem organizar suas tarefas e fazer anotações gerais, permitindo estudos, compromissos, eventos, tarefas de casa, hábitos, permitindo anexar arquivos, links e descrições dos mesmos.
 - Sistema de recompensas para incentivo da utilização da ferramenta
 - Diversidades e templates para melhor usabilidade
 - Relatórios de tempo gasto
